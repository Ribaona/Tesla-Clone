# Tesla Model 3 Landing Page

A modern, responsive landing page inspired by Tesla's official website, featuring the Model 3 and showcasing the entire Tesla vehicle lineup.

![Tesla Landing Page Preview](https://images.unsplash.com/photo-1551830840-48c31649dc6a?ixlib=rb-4.0.3&auto=format&fit=crop&w=2070&q=80)

## 📋 Project Overview

This project is a Tesla-inspired landing page that demonstrates modern web development practices with a focus on clean design, responsiveness, and user experience. The page features a hero section dedicated to the Tesla Model 3, a comprehensive models section showcasing Tesla Model 3 vehicle.

## ✨ Features

### Hero Section
- Full-screen background image of Tesla Model 3
- Eye-catching headline and call-to-action buttons
- "Touchless Delivery" promotional link
- Smooth scroll animations

### Models Section
- Tesla Model 3 display
- Model card includes:
  - High-quality vehicle image
  - Starting prices
  - Key specifications (Range, 0-60 mph, Top Speed)
  - Interactive "Order Now" buttons
- Hover effects and smooth transitions
- Featured highlight for Model 3


### Technical Features
- Fixed navigation bar with scroll effects
- External CSS architecture
- Semantic HTML5 structure
- Cross-browser compatibility

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom styling, animations, flexbox, grid
- **Google Fonts** - Montserrat font family
- **Unsplash API** - High-quality vehicle images
